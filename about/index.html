    我们正在维护的项目¶

    技术型¶

    主要以 JS 和 Python 为主，你也可以借着我们的平台锻炼一下你自己。

    ZUCC课程补全计划

    Exercises-Crawler，Chrome 插件，爬取网页中习题的工具，主要技术栈使用 HTML+CSS+JS。

    ZUCC课程补全计划

    Exercises-Manager-CLI，命令行程序，处理和翻译习题成 Markdown 格式，主要使用 Node.js。

    ZUCC课程补全计划

    Exercises-Manager-Desktop，跨平台桌面应用，为了方便部分不习惯命令行的用户，是 Exercises-Manager-CLI 的图形化界面，主要使用 Electron。

    ZUCC课程补全计划

    MarkWord，命令行程序，编写 Markdown 可以直接转 Word，方便平时写报告。
